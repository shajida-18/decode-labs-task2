# Decode Labs Internship - Task 2

A simple Backend API built using **Node.js** and **Express.js** as part of the Decode Labs Full Stack Internship.

## Features
- GET API endpoint
- POST API endpoint
- Basic input validation
- JSON request & response

## Tech Stack
- Node.js
- Express.js
- JavaScript

## Run the Project

npm install
node server.js


Server runs at:
http://localhost:3000
